# GenOIBlog
根据OI题目创建Markdown博客，给Hexo用。

不要以为这个可以读取PDF啥的。那么高级的我不会。我做的只是读进题目，加Header和Banner，然后写进文件。

## 正确使用方法：运行`genblog.py`，填写题目信息，然后大片复制粘贴。

![题目PDF](题目PDF.png)

### 2. 在博客目录下运行脚本，并且从题目中复制粘贴以下内容。
